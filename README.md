# Financial-Anomaly-Detection & Automated Reconciliation
# 财务异常检测与自动化对账系统
## Project Overview(项目概述）
This project is designed to solve the efficiency and accuracy challenges in traditional corporate accounting. By leveraging Python and Machine Learning, it automates the reconciliation process between MYOB/Xero exports and Bank statements, identifying risks that are pften missed by human eyes.

## key Features(核心功能）
    Automated Reconciliation: High-speed matching of GLentries and bank transactions, handling penny differences.
    (自动化对账：快速比对账面与流水，支持处理微小金额差异）
    Audit Rule Engine:  Automatically flags weekend transactions, duplicated invoices, and suspicious vendors.
    (审计规则引擎：自动标记周末交易、重复发票及可疑供应商）
    AI-powered Anomaly Detection:  Uses  Isolation Forest(ML)  to detect outtlier transactions without pre+defined thresholds.
    (AI异常检测：利用孤立森林算法检测离群交易，无需预测固定阈值）

## Tech Stack(技术线）
    Language: Python 3.x
    Libraries:  Pandas(data cleaning), Scikit-learn (Anomaly Detection).
    Domain Focus:  Corporate Finance, Internal Audit, Digital Transformation.

## Business Value(业务价值）
By inplementing this tool, a finance team can reduce manual reconciliation time by 80% amd increase audit coverage to  100% of transactions.
通过实施此工具，财务团队可将人工对账时间减少80%，并将审计覆盖率提升至100%的全量交易。
